## Estudo de algoritimos com portugol ##

Este repositorio foi criado para as minhas atividades práticas de lógica de programação utilizando a linguagem portugol(visuallg).

## Atividades ##

* CONVERSOR_F_C.ALG: conversor de temperatura de fahrenheit para celcius.
* EMPRESTIMO.ALG: algoritimo para calcular os juros de empréstimo.
* IDADE.ALG: vereficar a idade que você tem e verificando se é maior de idade com algoritimo condicional.
* IMPOSTO.ALG: calculo de imposto de de valores.
* REAIS-PARA-DOLAR.ALG: conversor de moedas(Real para dólar).

## atividades condicionais ##
* IDADE.ALG(é uma melhora para descobrir se é  de maioridade).
* PAR OU IMPAR.ALG: Descobrir se o número é impar ou par.
* IMC COM CONDICIONAL.ALG: calculo para saber se está acima do peso ou não.
* APTO À DIRIGIR.ALG: Descubra se está apto à dirigir.
* APROVADO OU REPROVADO.ALG: saber se o aluno foi aprovado atraves da medía de duas notas.
