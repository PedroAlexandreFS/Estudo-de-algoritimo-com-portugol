## Estudo de algoritmos com portugol ##

Este repositório foi criado para as minhas atividades práticas de lógica de programação utilizando a linguagem portugol (Visualg).

## Atividades ##

* CONVERSOR_F_C.ALG: conversor de temperatura de Fahrenheit para Celsius.
* EMPRESTIMO.ALG: algoritmo para calcular os juros de empréstimo.
* IDADE.ALG: verificar a idade que você tem e verificando se é maior de idade com algoritmo condicional.
* IMPOSTO.ALG: cálculo de imposto de valores.
* REAIS-PARA-DOLAR.ALG: conversor de moedas (Real para Dólar).

## Atividades condicionais ##
* IDADE.ALG (é uma melhora para descobrir se é de maioridade).
* PAR OU IMPAR.ALG: descobrir se o número é ímpar ou par.
* IMC COM CONDICIONAL.ALG: cálculo para saber se está acima do peso ou não.
* APTO À DIRIGIR.ALG: descubra se está apto a dirigir.
* APROVADO OU REPROVADO.ALG: saber se o aluno foi aprovado através da média de duas notas.
* NOTA A A F.ALG: Dertemina o desepenho da nota do aluno
* FUTEBOL.ALG: dertemina o resultado do jogo


## ATIVIDADE ESCOLHA CASO
* CRIANÇAESPERANÇA: algoritimo simulando doações
* SALARIO: Escolhando solario de acordo com depedentes
